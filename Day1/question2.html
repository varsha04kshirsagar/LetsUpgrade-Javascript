Var

var variable use for a variable declaration.Scope of var variable is globally or function/locally.The scope is global when a var variable is declared outside a function anf it is available for use in whole window.

Let

let is now preferred for variable declaration. let is block scoped. A block is a chunk of code bounded by {}. A block lives in curly braces. Anything within curly braces is a block.

Const

Variables declared with the const maintain constant values. const declarations can only be accessed within the block they were declared.
